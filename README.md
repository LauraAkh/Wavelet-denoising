# Шумоподавление изображений на основе вейвлет-преобразований

Проект выполнен в рамках научно-исследовательской практики на кафедре высшей математики РТУ МИРЭА. Посвящён сравнительному анализу вейвлетов первого и второго поколения для подавления аддитивного гауссовского и импульсного шума.

## Используемые технологии
- Python 3.8+
- NumPy, OpenCV, scikit-image
- PyWavelets
- Matplotlib (визуализация)

## Что сделано
- Реализованы схемы лифтинга для CDF 5/3 и медианного вейвлета (с нуля).
- Проведено сравнение с библиотечными вейвлетами (Symlet8, Coiflet2, Bior2.2) из PyWavelets.
- Качество восстановления оценивалось метриками PSNR и SSIM.
- Исследованы теоретические свойства: компактный носитель, нулевые моменты, симметрия, условия точного восстановления.

## Ключевые результаты
- Symlet8 показал наилучший PSNR (~26 дБ) для гауссовского шума.
- CDF 5/3 (собственная реализация) по качеству близок к библиотечным аналогам и подходит для сжатия без потерь (JPEG 2000).
- Медианный лифтинг устойчив к импульсному шуму, но сглаживает текстуры.
